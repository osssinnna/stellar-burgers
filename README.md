# Stellar burger

Веб-приложение для создания и заказа кастомных бургеров с использованием React, TypeScript и Redux Toolkit.4

## Деплой

## Основные возможности

- **Конструктор бургеров** - интерактивное создание бургера из доступных ингредиентов
- **Система заказов** - оформление и отслеживание заказов в реальном времени
- **Личный кабинет** - управление профилем и история заказов
- **Авторизация** - полноценная система регистрации и аутентификации
- **Лента заказов** - просмотр всех заказов в реальном времени
- **Адаптивный дизайн** - корректная работа на различных устройствах

## Технологии

- **React 18** + TypeScript
- **React Router**
- **Redux Toolkit**
- **CSS3**

## Установка и запуск

### Предварительные требования

- Node.js 18 или выше
- npm

## Запуск

Клонируйте репозиторий:

```bash
git clone https://github.com/osssinnna/stellar-burgers.git
cd stellar-burgers
```

Установите зависимости:

```bash
npm install
```

Запустите проект в режиме разработки:

```bash
npm start
```

Откройте http://localhost:4000 в браузере
